Some Introduction

In penetration testing, a lot of different tools are used. For our purposes, we’re going to be satisfied using nmap and metasploit.

This guide is going to show you some alternative ways of installing these tools. They are usually accessed from a Kali Linux distribution. This Kali Linux can be easily be setup from your Windows 10 or 11 with Windows Subsystem for Linux (WSL).

Kali Linux

Kali Linux is a penetration testing linux distribution often used for penetration testing. It have many standard tools already installed.

Nmap

Nmap is an enumeration tool, used to enumerate through ip addresses and ports. The goal here is often to find out what attack surface you have on a target.

Metasploit

Metasploit (msfconsole??) is a command line tool that contains a lot of vulnerabilities that are easy to use. It got everything from auxiliary scanners to working vulnerabilities you can use to exploit different servers.

Installation Guide

Kali Linux on WSL for Windows 10 / 11

There are many ways to get access to these tools on your computer. We recommend that you install WSL with Kali Linux, either with a GUI or the more regular way and accessing Kali from a terminal interface. Both ways are described below.

Some common problems.

Make sure you that you run powershell as administrator before you start installing.

A problem you might run into is that you need to enable hardware virtualization

https://learn.microsoft.com/en-us/virtualization/hyper-v-on-windows/about/ https://support.microsoft.com/en-us/windows/enable-virtualization-on-windows-11-pcs-c5578302-6e43-4b4b-a449-8ced115f58e1

Install Kali Linux

If you already have debian, ubuntu or some other linux machine set up, that should be sufficient. However Kali Linux is the usual distro for penetration testing so we will be installing this one.

Install WSL with Kali Linux

Instructions from kali.org: https://www.kali.org/docs/wsl/wsl-preparations/

Getting a Kali Linux GUI

This video show some similar steps, it should be easy enough to follow.

Install some regular dependencies

Kali Linux should come with a lot of tools fresh out of the box. But with WSL almost nothing will be installed. You might want to install some basic tools before starting:

sudo apt-get update -y
sudo apt-get upgrade -y
sudo apt-get install nmap -y
sudo apt-get install vim -y
sudo apt-get install git -y

Install nmap

From your now virtual (or real) kali linux, run the following command in terminal:

sudo apt-get install nmap -y

You can test that it works by running nmap google.com and getting something similar:
